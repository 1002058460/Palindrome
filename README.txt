Apa itu VSCode
	VSCode adalah software teks editor yang bisa menjalankan program JavaScript, HTML, CSS, dll.

Perbedaan Java & NodeJs
	Java adalah bahasa pemrograman yang dapat mengembangkan aplikasi mobile dan desktop. Javascript adalah bahasa pemrograman untuk membangun aplikasi web, biasanya digunakan di front end development. NodeJs adalah platform perangkat lunak yang digunakan untuk menjalankan Javascript diluar browser dan dapat diimplementasikan pada server, biasanya digunakan di back end development.
